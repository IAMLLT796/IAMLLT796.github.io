模拟题合集

用Java语言编写，假设你有一段SQL日志和Nginx日志混合在一起的文本数据，

格式大致如下:
2023-03-15 10:00:01 SQL: SELECT * FROM users WHERE id = 123;
2023-03-15 10:00:03 NGINX:GET /images/logo.png HTTP/1.1 200 OK
2023-03-15 10:01:15 SQL: UPDATE users SET name = ‘John’ WHERE id = 456;
2023-03-15 10:01:20 NGINX: POST /api/login HTTP/1.1 500 Internal Server Error
你的任务是编写一个解析这些日志并统计每种类型(SQL，NGINX)在每分钟内的发生频率的程序。输出结果应该是一个字典或其它数据结构，其中包含时间(精确到分钟)

示例输出:
‘2023-03-15 10:00’:{‘SQL’:1,’NGINX’: 1},
‘2023-03-15 10:01’:{‘SQL’:1,’NGINX’: 1}

额外要求:
请考虑程序的扩展性，以便未来可以轻松添加对新日志类型的解析。
请处理可能的异常情况，例如日志格式不正确或者日志类型不受支持编码问题:请提供一个函数，该函数接受日志文本字符串作为输入，并返回按分钟聚合的日志类型计数。
