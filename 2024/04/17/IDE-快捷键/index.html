IDE 快捷键

发表于2024-04-17更新于2024-04-17阅读次数

JetBrains 全家桶快捷键

新建

  • Alt + Insert

移动

  • Alt + 上下箭头：以函数为单位移动
  • Ctrl + G：定位到行
  • Ctrl + W：可以选择单词继而语句继而行继而函数
  • Ctrl + Shift + W：取消选择光标所在词
  • Ctrl + { 或 Ctrl + }：定位到大括号的开头和结尾

定位查找

  • Shift + Shift：查找任意文件
  • Ctrl + N：查找类
  • Ctrl + F：查找
  • Ctrl + R：替换
  • F4：进入某一方法或者变量定义处
  • Ctrl + F2：查看方法和成员变量
  • Ctrl + Alt + 左箭头（右箭头）：回退（前进）到上一次编辑处

编辑

  • Ctrl + D：复制一行
  • Ctrl + Y：删除一行
  • Ctrl + /：选中一段进行注释与反注释

简单重构

  • Shift + F6：重命名
  • Ctrl + Alt + M：抽取函数
  • Ctrl + Alt + V：抽取变量

简单调试

  • Ctrl + Shift + F9 / F10：调试/运行测试用例
  • Ctrl + F8：打断点/取消断点
  • Shift + F9 / F10：调试/运行
  • F7：进入函数
  • F8：单步
  • F9：到下一个断点或者结束
  • Alt + F8：查看表达式结果

其余

  • Ctrl + Shift + T：创建测试类
  • Ctrl + Shift + Enter：将光标移到下一行
  • Ctrl + Alt + V：创建变量
  • Home：从句尾移到句前
  • Alt + Enter：创建新方法
  • Alt + 1：代码区域切换到project窗口
  • Ctrl + E：回到上一个类
  • Ctrl + Alt + S：打开Setting
  • Ctrl + Alt + L：代码格式化
  • Alt + I：提取接口
  • Alt + U：重写方法
  • Ctrl + Alt + F：将new的对象重构成field
